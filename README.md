# book-inventory-system
Group project for implementing a digital book inventory system using Python and Django. Includes CRUD features, filtering by author, UML diagrams, and URL-based testing.
# Book Inventory System

This project is a digital inventory system for a local bookstore called **Readers Haven**.

## Problem Description
Readers Haven has been operating traditionally for 30 years. Due to the rise of online platforms, the owner wants to digitize the bookstore inventory to manage books more efficiently.

## Features
- Add books and authors to the inventory
- Retrieve the entire book inventory
- Filter and display books by a specific author

## Technologies Used
- Python
- Django
- SQLite

## How to Run the Project
1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Run migrations
5. Start the Django server
